EduNode
=======

A revolutionary holistic-education platform
-------------------------------------------

***Dependencies***

* (MongoDB)["http://mongodb.org"]
* (NodeJS)["http://nodejs.org"]
* (Redis)["http://redis.org"]

***Install***

* $ git clone --recursive 
* $ git remote add origin https://github.com/nextiamx/eduNode.git eduNode
* $ cd ./eduNode
* $ npm install
* $ node main.js
